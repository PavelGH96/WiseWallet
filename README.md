# WiseWallet v1.9

Личный учёт доходов, расходов и кредитов. Однофайловое PWA-приложение без сборки: работает на ПК и iPhone, офлайн, с опциональной синхронизацией через Supabase.

## Состав

| Файл | Назначение |
|---|---|
| `index.html` | всё приложение (HTML + CSS + JS) |
| `manifest.json` | PWA-манифест (установка на экран) |
| `sw.js` | service worker — офлайн-режим |
| `icon-192.png`, `icon-512.png` | иконки |
| `supabase_schema.sql` | схема таблицы синхронизации |

## Запуск

**Локально:** просто откройте `index.html` в браузере — всё работает без сервера (кроме установки PWA).

**GitHub Pages (рекомендуется):**
1. Создайте репозиторий на GitHub (можно приватный + Pages).
2. Загрузите все файлы в корень репозитория.
3. Settings → Pages → Deploy from branch → `main` / root.
4. Откройте выданный URL на iPhone → Поделиться → «На экран «Домой»» — получите полноценное приложение.

## Синхронизация Supabase (опционально)

1. Создайте бесплатный проект на supabase.com.
2. Выполните `supabase_schema.sql` в SQL Editor.
3. В приложении: Настройки → Синхронизация Supabase → вставьте URL проекта и anon key (Settings → API).
4. На втором устройстве укажите те же URL, key и **тот же ключ синхронизации**.

Синхронизация автоматическая: после каждого изменения и при восстановлении сети. Побеждает более свежая версия данных.
